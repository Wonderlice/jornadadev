# Exercício 2 — Pseudocódigo
**Escreva em pseudocódigo um algoritmo para cada item:**

## a. Calcular a área de um retângulo (base × altura)

```texto
Inicio
   leia base
   leia altura
   resultado ← base * altura
   escreva ( "A área do retângulo é: ", resultado)
Fim
```
## b. Verificar se um número é par ou ímpar

```texto
Inicio
   leia numero
   resto ← numero MOD 2
   se resto = 0 entao
      escreva ("O número é par")
   senao
      escreva ("O número é ímpar")
   fimse
Fim
```
## c. Encontrar o maior entre três números

```texto
Inicio
  leia num 1
  leia num 2
  leia num 3
  maior ← num 1
  se num 2 > maior então
     maior ← num 2
  fimse
  se num 3 > maior então
     maior ← num 3 
  fimse
  escreva("O maior número é: ", maior)
Fim
```
